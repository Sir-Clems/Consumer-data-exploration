# Consumer-data-exploration
This project focuses on exploratory data analysis (EDA) using Microsoft SQL Server to uncover meaningful insights from structured datasets. The objective is to analyze, transform, and summarize data to support data-driven decision-making.

Using SQL queries, the project explores key data characteristics such as distributions, relationships, trends, and anomalies. Core techniques applied include data filtering, aggregation, grouping, joins, and the use of built-in functions for calculations such as averages, totals, and date differences.

The dataset was examined to answer practical business questions, including performance evaluation, sales trends, and demographic insights. Queries were written to compute key metrics such as total sales, average values, and category-based summaries. Data cleaning steps were also performed where necessary to handle missing or inconsistent values.

Additionally, the project demonstrates the use of variables, parameters, and stored procedures to create reusable and dynamic SQL scripts. This improves efficiency and allows flexible querying based on different input conditions.

The goal of this exploration is not only to generate insights but also to showcase strong SQL skills in data manipulation, querying, and logical problem-solving. The results can serve as a foundation for further analysis, reporting, or visualization in tools such as Power BI or Tableau.
